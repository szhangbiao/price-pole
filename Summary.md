# 金融数据获取方式汇总

本文档汇总了项目中用于获取各类金融数据（三大指数、黄金价格）的接口方案，主要基于 **新浪财经 API** 结合 **Cloudflare Worker + KV 缓存** 的架构来实现高可用与无限制请求。

## 1. 架构设计与最佳实践

为了突破公共接口的频率限制，并保证服务的极高稳定性与低延迟，项目中采用以下架构：

*   **代理拉取层**：通过 Cloudflare Worker（例如 Cron Trigger 定时任务）去请求第三方公共接口。
*   **缓存层**：将获取并解析后的最新数据存入 Cloudflare KV。
*   **对外服务层**：前端与客户端只从 KV 读取数据，不直接请求公共接口。

这确保了第三方数据源**每天只受到极低频率的请求**，而对外则提供**不限次数、毫秒级响应**的 API。

## 2. A股三大指数获取方式

项目中已在 `src/service/priceService.tsx` 中实现了三大指数的获取逻辑。

*   **数据源 API**：`https://hq.sinajs.cn/list=[代码]`
*   **股票代码**：
    *   **上证指数**：`s_sh000001`
    *   **深证成指**：`s_sz399001`
    *   **创业板指**：`s_sz399006`
*   **请求示例**：`https://hq.sinajs.cn/list=s_sh000001,s_sz399001,s_sz399006`

### 请求头要求 (Header)
必须包含以下请求头，否则会触发新浪的反爬机制（403 Forbidden）：
```http
Referer: https://finance.sina.com.cn
User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 ...
Accept-Charset: GBK,utf-8;q=0.7,*;q=0.3
```

### 返回格式与解析
返回的是一段 JavaScript 变量赋值代码，且采用 **GBK 编码**（在 Cloudflare Worker 中必须使用 `new TextDecoder('gbk')` 进行解码）。
格式如下：
```javascript
var hq_str_s_sh000001="上证指数,3000.00,10.00,0.33,1000000,100000000";
```
按逗号 `,` 分割后的字段含义：
0: 指数名称 (如: 上证指数)
1: 当前点位 (如: 3000.00)
2: 涨跌额 (如: 10.00)
3: 涨跌幅百分比 (如: 0.33)
4: 成交量（手）
5: 成交额（万元）

## 3. 黄金价格获取方式

同样可以使用新浪财经的接口获取全球与国内黄金的实时走势。

*   **数据源 API**：`https://hq.sinajs.cn/list=[代码]`
*   **黄金代码**：
    *   **伦敦金（现货黄金）**：`hf_XAU`
    *   **国内黄金（沪金主连期货，走势与现货极度贴合）**：`AU0`
    *   **国内黄金ETF（如华安ETF作为现货参考）**：`sh518880`
*   **请求示例**：`https://hq.sinajs.cn/list=hf_XAU,AU0,sh518880`

### 请求头要求 (Header)
和指数一样，**必须**携带 `Referer` 头防拦截：
```http
Referer: http://finance.sina.com.cn
```

### 返回格式与解析
返回的数据也是 GBK 编码的逗号分隔字符串。

**伦敦金 (`hf_XAU`) 示例解析**：
```javascript
var hq_str_hf_XAU="4687.25,4715.070,4687.25,4687.60,4726.77,4676.34,21:16:00,4715.07,...";
```
(通常包含：当前报价、买价、卖价、最高、最低、时间等)

**沪金主连 (`AU0`) 示例解析**：
```javascript
var hq_str_AU0="沪金,145957,574.86,585.84,574.40,574.94,581.58,581.60,...";
```
(通常包含：品种名称、开盘价、最高价、最低价、最新价、买价、卖价等)

## 4. 其它备用方案 (精确 Au99.99 现货)

如果业务硬性要求获取绝对精确的上海黄金交易所 **Au99.99现货数据**（新浪对此支持较弱），可采用如下方式：
1. 注册聚合数据等付费 API 服务。
2. 结合 CF Worker 定时任务（如每 15-30 分钟拉取一次存入 KV），白嫖这些平台每日赠送的 100 次免费额度，将其通过 KV 缓存转换为不限次数的公共查询服务。
